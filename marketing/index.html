Run it with one command.

No install: bunx @waits/recallit runs the CLI. Point it at a source to generate an honest pack, then practice in your terminal. For the voiced browser session, clone the repo and bun run serve.

★ Star on GitHub Read the docs → 
# no install — run it with bunx
bunx @waits/recallit

# turn a PDF, URL, repo, or topic into a pack
bunx @waits/recallit pack <source>

# then practice it (needs ANTHROPIC_API_KEY)
bunx @waits/recallit daily
