Implementation sketch

POST /api/jobs

Returns { jobId, status: "queued" } immediately. No blocking. The chat shows the job card with that ID.

GET /api/jobs/:id

Polled every 5s. Returns { status, progress, result }. On "done" the FE injects the ledger into the message that owns this jobId.

useJobs() hook

A React context that owns all job state. Each useInterval polls active jobs. The tray and sidebar read from it. useChat stays separate.
